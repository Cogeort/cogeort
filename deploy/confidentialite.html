Politique de confidentialité.

La présente politique décrit la manière dont Cogeort collecte, utilise et protège vos données personnelles, conformément au Règlement général sur la protection des données (RGPD).

02Données collectées

Selon votre interaction avec le site :

  • Identification : nom, prénom, structure ;
  • Contact : e-mail, téléphone ;
  • Rendez-vous : date, créneau, sujet (via Calendly) ;
  • Navigation : adresse IP, navigateur, pages consultées.

Aucune donnée sensible n'est collectée.

03Finalités

  • Répondre à vos demandes ;
  • Organiser les rendez-vous ;
  • Établir des devis et exécuter les missions ;
  • Respecter les obligations comptables et légales.

04Bases légales

  • Consentement : prise de rendez-vous et échanges ;
  • Exécution d'un contrat : pour les missions ;
  • Obligation légale : pour la facturation ;
  • Intérêt légitime : sécurité du site.

05Durée de conservation

  • Prospects : 3 ans après le dernier échange ;
  • Clients : 5 ans après la fin de mission ;
  • Comptabilité : 10 ans ;
  • Logs : 12 mois.

06Partage des données

Cogeort ne vend ni ne loue vos données. Elles peuvent être traitées par les sous-traitants suivants :

  • Vercel Inc. — hébergement (USA, clauses contractuelles types) ;
  • Calendly LLC — prise de RDV (USA, idem) ;
  • Make (Celonis SE) — automatisation (UE) ;
  • Airtable Inc. — base de contacts (USA, idem) ;
  • LWS — nom de domaine (France).

07Cookies

Le site n'utilise aucun cookie publicitaire ni de mesure d'audience. Seuls les cookies strictement nécessaires à Calendly peuvent être déposés. Vous pouvez les bloquer via votre navigateur.

08Vos droits

Conformément au RGPD, vous disposez des droits suivants : accès, rectification, effacement, limitation, opposition, portabilité de vos données, et droit de définir des directives post-mortem.

Pour les exercer : thomas@cogeort.fr. Vous pouvez également introduire une réclamation auprès de la CNIL.
